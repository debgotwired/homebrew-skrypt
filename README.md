# Homebrew Tap for Skrypt

AI-powered documentation generator with code examples.

## Installation

```bash
brew tap debgotwired/skrypt
brew install skrypt
```

## Usage

```bash
skrypt --help
skrypt init
skrypt generate ./src
```
